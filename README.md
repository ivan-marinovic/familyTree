# familyTree

### Prerequisites
*Java 8+
*Maven 3.6 or later
*Git(if cloning repository)

### Installation
*clone repository : git clone https://github.com/ivan-marinovic/familyTree.git
*navigate to project directory
*run: mvn clean install
*run: mvn compile
*run: mvn package

### Usage
*run: java -jar target/familyTree-1.0-SNAPSHOT.jar

### Tests
*run: mvn test
